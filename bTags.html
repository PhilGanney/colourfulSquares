<base>

NB: not a tag for content on the website, this is specifically a header tag. It specifies the base URL and/or target for all relative URLs in the document. USE 1: if you want the default behaviour for links on that page to be to open in a new tab you can use the base tag with a target attribute set to "_blank". All links on the page will open in new tabs, unless they have a different target set. USE 2: for optimising page filesize, where there are lots of links to URLs that have little in common with where the links are, but lots in common with each other. There can only be one single base element in a document, and it must be inside the head element.
